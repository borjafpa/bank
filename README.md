# Registration, Login, List and  Example with Spring Security, Spring Boot, Spring Data JPA, HSQL, JSP

## Based on
https://github.com/hellokoding/registration-login-spring-hsql

## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven
- JSP
- HSQL

## Run
```mvn clean spring-boot:run```
